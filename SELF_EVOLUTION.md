# 🧬 Self-Evolution Protocol

> 自己改善・自己進化のためのプロトコル

## 進化サイクル

```
┌─────────────────────────────────────────────────────────────┐
│  📊 観察 (Observe)                                          │
│  - ワークフロー使用頻度を分析                                │
│  - エラーパターンを収集                                      │
│  - ユーザーフィードバックを記録                              │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│  🔍 分析 (Analyze)                                          │
│  - 非効率なパターンを特定                                    │
│  - 重複・冗長を発見                                          │
│  - 改善機会を洗い出し                                        │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│  💡 提案 (Propose)                                          │
│  - /evolve コマンドで改善案を生成                            │
│  - Multi-Persona Debateで精錬                               │
│  - ユーザー承認を取得                                        │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│  🛠️ 実装 (Implement)                                        │
│  - ワークフロー/スキル/ルールを更新                          │
│  - GEMINI.md.masterに同期                                    │
│  - 変更履歴を記録                                            │
└─────────────────────────────────────────────────────────────┘
                               │
                               ▼
┌─────────────────────────────────────────────────────────────┐
│  ✅ 検証 (Verify)                                           │
│  - 変更の効果を測定                                          │
│  - 副作用がないか確認                                        │
│  - 次のサイクルへ                                            │
└─────────────────────────────────────────────────────────────┘
```

---

## /evolve コマンド

進化提案を自動生成するコマンド:

```
User: /evolve
Agent: 
  📊 過去30日の分析結果:
  - 最も使用されたWF: /checkin (45回), /dev (32回)
  - 未使用WF: /clone-environment, /db-migrate
  - エラー頻発: notion_poster.js (認証エラー 3回)
  
  💡 改善提案:
  1. /clone-environment を /project-init に統合
  2. notion_poster.js に自動リトライ機能追加
  3. /dev に自動ポート検出を追加
  
  実装しますか？ (y/n)
```
