## 🎯 5軸分解

| 軸 | 内容 |
|---|---|
| Target | AIが人間の言及なしに、自律的にメタルール（MR-01〜09）を判断プロセスに適用できる状態 |
| Core Tension | 「コンテキストに追加する」と「常に適用する」は別物。読み込んでも使われなければ意味がない |
| Risk | GEMINI.mdに参照を追加するだけの表面的な解決になる / 実装後も同じ問題が起き続ける |

| Success Criteria | 人間が言及しなくても、AIがセッション中にメタルールを自律的に適用して判断・行動を変えていること |
