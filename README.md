## 🎯 これは何？

Antigravityは、AIエージェントがあなたの開発チームになる環境です。

**あなたがやること**:
- ✅ アイデアを考える
- ✅ AIに指示を出す
- ✅ 作品を公開する

**Antigravityがやること**:
- ✅ 環境セットアップ
- ✅ ワークフロー自動化
- ✅ GitHub同期
- ✅ エラー修正
- ✅ デプロイ支援

---

## 🚀 5分でセットアップ

```bash
curl -sL https://raw.githubusercontent.com/RYKNSH/antigravity-core/main/setup.sh | bash
```

これだけで完了です。

## 📁 構造

```
~/.antigravity/               # Antigravity本体（GitHub管理）
├── agent/
│   ├── workflows/            # 35+ ワークフロー
│   ├── skills/               # 12+ スキル
│   ├── scripts/              # 27+ スクリプト
│   └── rules/                # ルール
├── knowledge/                # 13+ ナレッジベース
├── contrib/                  # コミュニティ貢献
└── personal/                 # 個人用（.gitignore）
```

---

## 🎓 使い方

### セッション開始
```bash
/checkin
```

### 作業
```bash
/go "ログイン機能を追加"
```

### セッション終了
```bash
/checkout
```

**これだけです。**
