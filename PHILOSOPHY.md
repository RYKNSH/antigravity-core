# なぜAntigravityを作ったか

## 問題意識

**エンジニアリングの壁が、創造性を阻んでいる。**

私たちは、素晴らしいアイデアを持つ人々が、技術的な障壁によって実現を諦める姿を何度も見てきました。

- 「アプリを作りたいけど、コードが書けない」
- 「ノーコードツールでは限界がある」
- 「エンジニアを雇う予算がない」
- 「学習する時間がない」

これらは、もはや言い訳であってはなりません。

## AI時代の可能性

**ChatGPT、Claude、Cursorの登場で、世界は変わりました。**

「コードが書けない」人でも、AIと対話することで、驚くべき成果物を生み出せるようになりました。

しかし、まだ壁があります：

- 開発環境のセットアップ
- ワークフローの管理
- バージョン管理
- デプロイ
- エラー対応

**これらの「エンジニアリング作業」が、創造性を阻んでいます。**

## Antigravityのビジョン

**AI時代には、誰もがクリエイターになれる。**

Antigravityは、AIエージェントがあなたの「開発チーム」になる環境です。

あなたがやること：
- ✅ アイデアを考える
- ✅ AIに指示を出す
- ✅ 作品を公開する

Antigravityがやること：
- ✅ 環境セットアップ
- ✅ ワークフロー自動化
- ✅ GitHub同期
- ✅ エラー修正
- ✅ デプロイ支援

**あなたは「何を作るか」に集中できます。**

## 設計思想

### 1. 非エンジニアファースト

Antigravityは、エンジニアのためのツールではありません。

**ターゲット**: AI-First Creators
- ChatGPT/Claudeで作品を作っている
- コードは読めるけど書けない
- ノーコードツールに限界を感じている
- 「もっと本格的に作りたい」と思っている

### 2. AI-Native

Antigravityは、AI時代のために設計されています。

- ワークフローは「AIエージェント」が実行
- ドキュメントは「AIが読みやすい」形式
- エラーは「AIが修正」できる構造

### 3. オープンソース

Antigravityは、コミュニティのものです。

- **AGPL v3**: 真のオープンソース
- **完全無料**: 個人・商用問わず
- **貢献歓迎**: ドキュメント、翻訳、バグ報告

### 4. ポータブル

Antigravityは、どこでも動きます。

- **GitHub同期**: 複数PC対応
- **5分セットアップ**: 新PCでも即座に
- **SSD不要**: クラウドベース

## 誰のため？

### メインターゲット: 「AI-First Creators」

**具体例**:
- 「Cursorで個人アプリを作ってる大学生」
- 「AIでブログ自動化してるマーケター」
- 「ノーコードツールに限界を感じたデザイナー」
- 「ChatGPTで副業アプリを作りたい会社員」

### 歓迎する人:
- 初心者エンジニア（学習中）
- 好奇心旺盛な人（仕組みを知りたい）
- オープンソース貢献者

### 推奨しない人:
- プロエンジニア（より高度なツールがあります）
- 大規模チーム（エンタープライズ向けではありません）

## ポジショニング

Antigravityは、**ノーコードツールの制限**と、**プロ開発環境の複雑さ**の間にある、**第三の選択肢**です。

```
ノーコード          Antigravity         プロ開発環境
(制限が多い)     (バランス)          (複雑すぎる)
    ↓                ↓                   ↓
  Bubble          Antigravity          VS Code
  Webflow         + AI Agent           + Docker
  Notion                               + Kubernetes
```

## 未来への期待

Antigravityは、新しいクリエイター階級を生み出します。

- **バイブコーダー**: コードは書けないけど、AIで作品を生み出す人
- **AI-First Creators**: AIを第一のツールとして使うクリエイター
- **ノーコード卒業生**: ノーコードツールを卒業し、次のステップへ

**これは、ムーブメントです。**
