## なぜ、MacBookではダメなのか？

MacBook Proは素晴らしいマシンです。しかし、物理的な制約からは逃れられません。

- **熱**: 高負荷な処理をするとファンが回り、膝上が熱くなる。
- **バッテリー**: 本気を出せば数時間で尽きる。結局、重い充電器が必要になる。
- **盗難リスク**: 50万円のマシンをカフェに置く緊張感。

対して、自宅に置かれた「Mac mini」はどうでしょう？
有線LANの安定性、無尽蔵の電力、静寂なファンレス（M4）。
この **「固定砲台」のパワーを、iPadという「魔法の板」から引き出す** ことができれば、全ての悩みから解放されます。

## 3種の神器：不可視のLANケーブルを作る

この環境を実現するために必要なのが、以下の3つのツールです。

### 1. Tailscale：魔法のLANケーブル

「家の外から自宅のMacに繋ぐ」
ネットワークエンジニアなら、これがどれほど面倒なことか知っているはずです（ポート開放、固定IP、セキュリティ...）。

しかし、Tailscaleを使えば **「アプリを入れてログインするだけ」** で終わります。
これは、世界中どこにいても、あなたのiPadとMacを **「同じ部屋にある」** ことにしてくれる魔法のVPNです。
カフェのWi-Fiからでも、4Gテザリングからでも、安全な暗号化トンネルで自宅のMacに直結します。

### 2. Jump Desktop (Fluid)：指をマウスに変える技術

リモートデスクトップには「遅延」と「操作のしにくさ」がつきものでした。
しかし、Jump Desktopの **Fluidプロトコル** は別次元です。
4G回線ですら驚くほど滑らかに動き、iPadのタッチ操作を完璧にMacのカーソル操作に変換します。

特に重要なのが **「3本指ドラッグ」** の設定。
これさえあれば、トラックパッドがなくてもテキスト選択やウィンドウ移動がストレスなく行えます。

### 3. BetterDisplay：ニセモノの画面を作る

Mac miniにはディスプレイが繋がっていない（ヘッドレス）ことが多いですが、そのままだとGPUが映像出力をサボり、画面がカクついたり解像度がおかしくなったりします。

そこで、BetterDisplayを使って **「仮想の超高解像度ディスプレイ」** を作成します。
iPad Pro 12.9インチなら `2732x2048 (HiDPI)` のダミー画面を作ることで、iPadのRetinaディスプレイにドットバイドットで美しく表示させることができます。
これで「リモート感」は完全に消え失せます。

## トラックパッドを捨て、AIの指揮者になる

「でも、iPadの画面じゃ細かいコード修正とか無理でしょ？」

そう思うかもしれません。私も最初はトラックパッドを持ち歩こうとしていました。
しかし、AI時代にその考えは古いです。

**Vibe Coding (没入コーディング)** の真髄は、**「自分で書かない」** ことです。

1.  **Jump Desktop** で自宅のMacに接続。
2.  **Cmd + I** で Cursor (AIエディタ) の全画面入力モードを開く。
3.  iPadのマイクに向かって、やりたいことを喋る。

「この関数のエラーハンドリングを追加して」
「デザインをもっとモダンにして」

あなたがやるべきは、1文字ずつタイプすることではなく、**AIに的確な指示を出す（指揮する）** ことだけ。
これなら、タッチ操作と音声入力だけで十分です。

## 永遠に切れない接続を作る

最後に、再起動しても接続が切れない「不死の設定」を施しておきましょう。
ターミナルで以下のコマンドを一回打つだけです。

```bash
# 停電復帰とWake on LANを有効化
sudo pmset -a autorestart 1 womp 1 screen 0
```

さらに、Jump Desktop Connectの設定で **「Launch at login」** をオンにしておけば、Macを再起動した直後（ログイン画面）からiPadで接続できます。
これで、あなたのMacは24時間365日、どこからでもアクセス可能なプライベートクラウドになりました。
