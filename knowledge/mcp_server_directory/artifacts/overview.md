# MCP Server Directory & Configuration

## 概要

このナレッジアイテムは、Model Context Protocol (MCP) サーバーのカタログと、主要なクライアント（Cursor, VS Code, Claude Desktop）への設定ガイドを提供します。

MCP を使用することで、AIアシスタントが直接外部ツールやデータソース（GitHub, Railway, Supabase 等）を操作できるようになります。

## 構成

- [クライアント設定ガイド](./clients/setup_guides.md): 各エディタ/アプリでの導入方法
- [インフラ・デプロイ](./directory/infrastructure.md): Railway, Supabase などの管理ツール
- [AIモデル・メディア](./directory/ai_and_media.md): OpenAI, Anthropic, Gemini, Discord, ElevenLabs
- [開発ツール](./directory/development_tools.md): GitHub, Filesystem, Git
- [生産性ツール](./directory/productivity_tools.md): Notion, Google Drive
