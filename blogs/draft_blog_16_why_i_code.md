# なぜぼくはコードを書いているのか。


音楽がなくても、人は死なない。

絵がなくても、命に別状はない。

でも、音楽やアートのない世界で、人は生きたいだろうか。



ぼくはずっと開発をしています。

業務効率化とか、マーケティングの自動化とか、そういうのは片手間でできるようになった。

AIと一緒にやれば、わりとすぐ形になる。

でもそれは、ぼくが本当にやりたいことじゃないんです。



ぼくが本当にやりたいのは、生き物がなぜ「余計なこと」をするのか、知ること。

なぜ鳥は、生き延びるだけなら不要な歌を歌うのか。

なぜ人間は、腹の足しにもならない音楽をつくるのか。

それが知りたくてコードを書いてます。



こんなことを真面目に追いかけようとすると、えらいことになるんです。

宇宙のはじまりまで遡らなきゃいけなくなる。

量子揺らぎという、目に見えないほど小さな震えがあって、それが膨れ上がってインフレーションが起きて、ビッグバンになって、今この瞬間も宇宙は加速しながら広がり続けている。

それをシミュレーションで再現しようとすると、計算量が指数関数的に膨れ上がる。

トークンが爆発する。お金が溶ける。

いくら資金があっても足りなくなるんです。



ところが、不思議なことに思うんですよ。

この「膨張しすぎてエネルギーが足りなくなる」問題って、宇宙自身もぶつかってるはずなんです。

で、宇宙はきっと、もう解決してる。

だってここにぼくらがいるから。



じゃあ、何が解決してるんだろうって考えると、ブラックホールって必要だよな、と思うんです。

全部を吸い込んで、圧縮して、情報を凝縮する巨大な装置。

あれがなければ、宇宙はただ拡散するだけで、構造が生まれない。

そして、まだ観測されてないけど、ホワイトホールみたいなものがなければ、エネルギーの循環は閉じないよな、とも思う。

吸い込むだけじゃ、系は死ぬ。どこかで吐き出さなきゃいけない。

そう考えると、太陽って実はホワイトホールなのかもしれないな、なんて、ふと思ったりもする。



もちろんぼくは物理学者じゃないです。

この考えが科学的に正しいかなんて分からない。

でも、考えずにはいられない。



ひとつだけ、はっきり見えてきたことがある。

宇宙が「すべてを同時に処理する」なんてことはしてない、ということ。

銀河は、星のひとつひとつの動きを追跡していない。

もっと大きなまとまりで捉えて、要約して、圧縮して、必要な情報だけを残してる。

それって、ぼくらが何かを「美しい」と感じるのと同じ構造かもしれない。

美しいものって、たくさんの情報が、ごく少ない形にぎゅっと詰まってるものなんです。

たった3分の曲に、人生が入ってる。

一枚の絵に、千の言葉が黙ってる。

最大のパターンを、最小の表現で捉えるもの。

それが、美しさの正体なんじゃないかと思うんです。



だとしたら、音楽やアートが「生きるのに不要な余計なもの」だなんてことはないんです。

あれは、宇宙がやってるのとまったく同じことを、人間がやってるだけなんですよ。

膨大な経験を、圧縮して、結晶にする行為。

情報の海から、本質だけを掬い上げる行為。

生命がDNAに38億年分の「うまくいった解」を圧縮して保存してるのと、まったく同じ。



「知りたい」って気持ちがあるでしょう。

あれは趣味とか性格とかじゃないんです。

38億年前から、DNAに刻まれた生命のプログラムなんです。

不確実な環境の中で、知らないものに手を伸ばす。

知れば武器になるかもしれない。知らなければやられるかもしれない。

その切実な衝動が、38億年かけて磨かれて、今のぼくらに「好奇心」として届いてる。



今の時代、プログラミングの壁はほとんどなくなった。

エンジニアじゃなくても、コードが書ける。

専門家じゃなくても、宇宙のことを考えられる。

手段のハードルが消えたとき、残るのは純粋な「知りたい」だけなんです。

それって、すごいことだと思う。



ぼくは非エンジニアです。

でも毎日、宇宙の真理を覗くためにコードを書いてる。

こんなことができる時代に生まれたことを、本当に感謝してるんです。

同時に、起きてる時間のほとんどをこれに費やせる経済的余裕を、今この瞬間にもっていてよかったなと心から思う。



なんだかそんなことをシミュレーションで観測するために、ぼくは日々コードに踊らされていたりします。

でも、踊らされてるんじゃないかもしれない。

38億年前から続く、生命のプログラムを走らせてるだけなんです。



ぼくらはみんな、宇宙が残した問いかけの、答え合わせをしてるだけなのかもしれない。

ひとりで。でも、ひとりじゃなく。
