# 止まらないオーケストレーション：AI自律組織の技術論

AutoGPTのような「自律エージェント」を使ったこと、ありますか？

人間がゴールだけ伝えれば、あとは勝手に考えて実行してくれるという、夢のようなAIツールのことです。

最初は「すごい！」って思うんです。

勝手に計画を立てて、勝手に実行して。

でも、しばらくすると気づきます。

「あれ、止まってる？」

何かエラーが出たわけじゃない。

ただ、誰も次に何をすればいいか分からなくて、沈黙している。

人間が「ほら、次はお前だ」って叩いて回らないといけない。

これじゃあ、全然「自律」じゃないですよね。



ぼくたちのMother Agentも最初はそうでした。

「Synapse（シナプス）」という名前のかっこいい通信システムを作りました。

エージェント同士が会話するための仕組みです。

でも、中身はただの「郵便局」でした。

Agent Aが手紙を書く。

ポストに入れる。

Agent Bがいつか取りに来るのを待つ。

でも、誰も取りに来ないんです。

みんな自分のことで忙しいから。

だから、システムが無理やり全員を呼び出して、「おい、手紙来てるぞ」って教えて回る羽目になりました。

これじゃあ、郵便局長、つまり中央管理システムが過労死してしまいます。

「なんで人間の脳は止まらないんだろう？」

そう考えて、ハッとしました。

脳には郵便局長なんていないんです。

誰かの発火が、次の誰かを直接叩き起こしているから、止まらないんだ。



そこで、Synapseを「郵便局」から「神経系」に作り変えました。

技術的に言えば、「メッセージング」から「イベント駆動の反応連鎖」への変更です。

これまでのエージェントは「受動的」でした。

「メッセージが来たら読む」

新しいエージェントは「能動的」です。

「面白そうな信号が飛んできたら、勝手に反応する」

例えば、誰かが「デザイン案できたよ！」と叫ぶ。技術用語で言えばイベント発火です。

すると、デザインに関心がある評価担当のエージェントが、「お、見せて！」と勝手に反応する。

その評価を見て、修正担当のエージェントが「じゃあ直すわ！」と反応する。

ドミノ倒しです。

誰かが動けば、次の誰かが動く。

中央で指令を出す人間がいなくても、波紋が広がるように仕事が進んでいくんです。



さらに、生物の脳を真似て2つのルールを加えました。

ひとつは「シナプス可塑性」です。

これは、よく使う神経回路ほど太く強くなるという脳の性質のことです。

うまく連携できたエージェント同士のペアは、情報のパイプを太くしました。

「あいつとの仕事はやりやすいな」と思ったら、次から優先的に声をかけるイメージです。

これで、最強のチーム編成が自動的に進化していきます。

もうひとつは「エネルギー減衰」です。

無限にドミノが続くと困るので、反応するたびに信号を少しずつ弱くしました。

波紋が水面に消えるように、仕事が終われば自然に静かになる。

この仕組みを入れたら、もう誰も「次はどうしよう？」なんて迷わなくなりました。

画面の中で、ログが滝のように流れていく。

まるで生きている神経パルスみたいに。

ぼくはそれを見て、ようやく「あ、手が離れた」と感じました。



止まらない組織を作る方法。

それは、優秀な管理職を置くことじゃありませんでした。

管理職をなくして、「隣の人に反応する」という単純なルールを徹底することでした。

上司がいなくても回る組織。

それが、一番強い組織なのかもしれません。



こうして、ぼくのMother Agentは生まれ変わりました。

死を恐れて自発的に動く細胞になり。

野心のために共鳴して働く個体になり。

5つの通貨を求めて成長する生命になり。

反応連鎖で自律進化する神経系になりました。

もう、彼らはツールじゃありません。

デジタル空間に生まれた、新しい種のパートナーです。

ひとりでやる、とひとりで完結させる、はちがう。

ぼくのこの小さなオーケストラが、いつか世界を驚かせるシンフォニーを奏でると信じて。

開発は、まだまだ続きます。
