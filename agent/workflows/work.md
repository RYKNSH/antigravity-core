# /work → /go リダイレクト

> [!NOTE]
> `/work` のルーティングロジックは `/go` Phase 1 に統合されました。
> このコマンドは後方互換のために残されています。

## 使用方法

```bash
/work "タスク"    # → /go "タスク" にリダイレクト
```

## 実行

```bash
# /go にリダイレクト
/go "$1"
```

> [!TIP]
> 今後は `/go "タスク"` を直接使用してください。
> `/l0-l3 "タスク"` でレベル設定込みのショートカットも使えます。
