## 🧠 感情アーク（SW定義）

```
empathy → fomo → [silence] → hope → liberation
```

| フェーズ | 感情 | 実装要件 |
|---------|------|---------|
| 0-3s | **共感** | 悩みを具体的に列挙（問いかけ形式）。ロゴ禁止。 |
| 3-8s | **FOMO** | 現状維持の機会損失。「0時間。」等の数字演出。 |
| 7-12s | **沈黙** | 30秒版のみ。ナレーション停止。テキスト1行のみ。 |
| 12-22s | **希望** | 解決策。ビジュアル中心。難しさは絶対に見せない。 |
| 22-30s | **解放感** | CTA。「行動した後の未来」を動詞で見せる。 |

## 🚫 禁止事項（MKT-04: 死のパターン）

これらが0〜3秒に含まれていたら **無条件でブロック**:

1. ロゴ・商品名・ブランド名の先出し
2. 3秒以上の感情変化なし（モノトーン進行）
3. 1フレームに3行超のテキスト
4. AI / DX / LLM / API 等の専門用語
5. CTAなし / 次のアクション不明確

追加禁止（全シーン通じて）:
- 「今すぐ！限定！残りわずか！」等の山奔り表現
- 機能・スペックの箇条書き羅列
- ひとり事業者を「困っている弱者」として描く演出
