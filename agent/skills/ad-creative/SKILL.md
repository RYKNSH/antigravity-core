## Phase 1: フレームワーク適用（THINK Gate必須）

### 構造フレームワーク: Hook → Retain → Reward (H→R→R)

```
0–3s   HOOK      スクロールを止める。ペインポイントで自分事化。
3–10s  RETAIN    オープンループ。前方テンションを構築。
10–15s REWARD    マイクロインサイト + CTA。摩擦低減型。
```

### Hook設計（5類型 — 優先順）

| # | 類型 | 形式 | 効果 |
|---|------|------|------|
| 1 | Pain Point First | 「〜してませんか？」→ 即座に自分事化 | CTR最高 |
| 2 | Curiosity Gap | 不完全情報 → 解決に視聴が必要 | 視聴完了率最高 |
| 3 | Pattern Interrupt | 予想外のビジュアル/サウンド | スキップ率-25% |
| 4 | Counterintuitive | 常識を否定する主張 | 認知的不協和→注目 |
| 5 | Social Proof | 具体的数値の実績 | 信頼+FOMO |

### 必須ルール

**H-1**: ロゴ・ブランド名で始めない。視聴者のペインが先。
**H-2**: 3秒ごとに視聴継続の理由を与える（3秒ルール）。
**H-3**: テキストは1フレーズ5-7語MAX。1秒以内に読めること。
**H-4**: アクセントカラーは1色1機能（最重要3語に限定使用）。
**H-5**: アニメーションスタイルは3種類MAX（フェードイン/スケール/スライド等）。
**H-6**: テキスト配置: 上1/3〜中央。下20%はUI重複ゾーンで使わない。
**H-7**: CTAは摩擦低減型。「無料で受け取る」>「詳しくはこちら」。具体的に何を得るか明示。
**H-8**: サウンドなしでもメッセージが完結すること（68%がミュート視聴）。
**H-9**: spring/ease-out カーブ使用。linear禁止（安っぽく見える）。
**H-10**: 1広告1メッセージ。複数の訴求を詰め込まない。

### コスト比較ベンチマーク（ソロプレナー/コーチング/高単価）

| 指標 | 弱い | 普通 | 強い |
|------|------|------|------|
| Hook Rate (3s視聴/imp) | <25% | 25-35% | 35%+ |
| CTR (Meta) | <1% | 1.67% | 2.5%+ |
| LP→登録率 | <20% | 20-30% | 30-40% |

### COPY_RULES との統合

- 煽りワード禁止（残席わずか/今だけ/革命的）→ 事実ベースの具体性で代替
- 収益系禁止（月収/年収/稼ぐ）→ 時間回収・タスク消滅で表現
- マーケ臭ワード禁止 → 「伝える仕組み」「あなたの言葉が届く状態」
- CTAに「!」禁止 → 静かな確信のトーン

### アンチパターン（絶対やらないこと）

- AP-1: ロゴ/ブランド開始（注意を獲得する前にブランドを出す）
- AP-2: 美しいが伝わらない（演出>メッセージ）
- AP-3: 釣りフック（過剰な期待→失望→信頼喪失）
- AP-4: 複数メッセージ（15秒に2つ以上の訴求）
- AP-5: 摩擦のあるCTA（「詳しくはこちら」は曖昧すぎる）
- AP-6: 無音非対応（テキストだけでメッセージが完結しない）

## Phase 2: 実装

### テンプレート3種の設計意図

| テンプレート | H→R→R での役割 | Hook類型 |
|-------------|---------------|---------|
| HookAd | Pain Point First → オープンループ → CTA | Type 1 (Pain) |
| ProofAd | Social Proof → Authority → CTA | Type 5 (Proof) + Type 2 (Curiosity) |
| DemoAd | Pattern Interrupt → BAB構造 → CTA | Type 3 (Interrupt) |

### 実装チェックリスト

- [ ] 各テンプレートがH→R→R構造に従っている
- [ ] 最初の3秒にペインポイントまたはパターンインタラプトがある
- [ ] テキストが5-7語MAX
- [ ] アクセントカラーが最重要語句のみに使われている
- [ ] アニメーションが3スタイル以内
- [ ] CTAが具体的（何を得るか明示）
- [ ] サウンドなしでメッセージが完結する
- [ ] spring/ease-outカーブを使用
- [ ] 下20%にテキストがない

## Phase 3: 検証

```bash
cd ads/remotion && npx remotion studio   # プレビュー
```

### 検証基準

1. **3秒テスト**: 0-3秒だけ見て視聴継続したいか？
2. **ミュートテスト**: 音なしでメッセージが伝わるか？
3. **1メッセージテスト**: 何の広告か1文で言えるか？
4. **CTA明確性テスト**: 何を得られるか即座にわかるか？
